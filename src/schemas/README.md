# Schemas

Esta pasta contém os schemas de validação da aplicação.

## Propósito

- Validação de dados de entrada
- Definição de regras de validação
- Transformação de dados
- Documentação de estruturas de dados

## Convenções

- Use bibliotecas de validação consistentes
- Documente as regras de validação
- Mantenha os schemas reutilizáveis
- Siga as convenções de nomenclatura do projeto
- Valide dados de entrada e saída
